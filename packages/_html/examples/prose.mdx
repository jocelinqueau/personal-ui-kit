# What a prose is made of ?

## What is prose ?

Prose is a class in tailwind in order to write content. It help creating nice layout of text that break line around the 70 characters.
Why 70 characters ? Because it is the optimal number of characters for a line of text to be readable. You can find the tailwind plugin [here](https://github.com/tailwindlabs/tailwindcss-typography)
